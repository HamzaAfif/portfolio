Automatisation de Facturation & Simulation de Crédit

Streamlined invoicing and credit simulations using Excel VBA and Python (Tkinter & OpenPyXL).

Explore Features

Project Overview

This project combines Excel VBA and Python to create a robust solution for managing invoicing and credit simulation. It integrates client and product management, invoice generation, and real-time credit simulation for financial planning.

Key highlights:

  • Automated invoicing and credit simulation using Excel VBA and Python.
  • Interactive GUIs for adding clients and products (Tkinter).
  • Generated invoices in Word and PDF formats for easy sharing and storage.
  • Simulated credit payments with detailed amortization schedules.

Key Features

  • Excel VBA: Automated invoice creation and credit simulations directly in Excel.
  • Python Tkinter: Built a GUI for managing clients, products, and invoices.
  • OpenPyXL: Programmatically handled Excel files for seamless data integration.
  • Dynamic Data: Real-time updates and feedback displayed in user-friendly forms.
  • PDF Generation: Exported invoices to PDF format for sharing and record-keeping.
