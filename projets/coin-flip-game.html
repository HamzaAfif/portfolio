Project Overview

The Coin Flip Game is a decentralized blockchain-based game built using Solidity, Truffle, Ganache, and Web3.js. It allows two players to join, flip a coin, and compete for a prize pool. The game features role switching, secure transactions via MetaMask, and a dynamic prize pool system.

Key features include creating and joining games, making guesses, flipping the coin, and leaving the game. The game ensures fairness and transparency through blockchain technology.
