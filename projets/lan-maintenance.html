LAN Network Maintenance

A detailed project to configure, analyze, and optimize LAN networks using Cisco tools, GNS3, and Wireshark.

Explore Features

Project Overview

The LAN Network Maintenance project aimed to simulate, configure, and maintain a Local Area Network (LAN). Using tools such as Cisco Packet Tracer, GNS3, and Wireshark, the project involved analyzing traffic patterns, configuring hardware, and troubleshooting connectivity issues.

The project focused on the following key objectives:

  • Designing and simulating LAN networks with logical and physical topologies.
  • Configuring routers, switches, firewalls, and servers for efficient traffic management.
  • Analyzing network traffic with Wireshark to monitor and optimize network performance.
  • Simulating an ARP spoofing attack using Yersinia and analyzing its impact on the network.
  • Setting up Active Directory and a Windows Server environment for centralized user and resource management.
  • Configuring DNS servers for resolving domain names within the LAN environment.

Key Features

  • Simulation Tools: Used Cisco Packet Tracer and GNS3 to create and test logical and physical network diagrams.
  • Hardware Configuration: Configured Cisco ISR 4331 router, ASA 5505 firewall, and Catalyst switches.
  • DHCP and DNS Servers: Set up DHCP servers for automated IP address allocation and DNS servers for resolving domain names.
  • Network Security: Simulated ARP spoofing attacks using Yersinia to evaluate security vulnerabilities and implemented firewall rules to mitigate risks.
  • Windows Server and Active Directory: Deployed a Windows Server environment with Active Directory for centralized user management and resource allocation.
  • Packet Analysis: Tracked and analyzed packets using Wireshark to optimize performance and identify bottlenecks.

Technical Details

The LAN Network Maintenance project demonstrates advanced network management skills through the use of various tools and technologies. The technical stack includes:

  • Cisco Packet Tracer: Simulated the network's logical and physical layouts for training and troubleshooting.
  • GNS3: Emulated real-world network environments to test router and switch configurations.
  • Yersinia: Simulated ARP spoofing attacks to understand vulnerabilities and security threats.
  • Wireshark: Captured and analyzed packet flows to optimize LAN traffic and troubleshoot issues.
  • Windows Server: Deployed Active Directory for centralized user and resource management within the LAN environment.
  • DNS Configuration: Configured DNS servers to resolve internal domain names and ensure seamless communication within the LAN.
