Chat with CSV

An innovative application that leverages AI and CSV data to provide actionable cybersecurity insights.

Explore Features

Project Overview

"Chat with CSV" bridges the gap between data and AI by enabling users to query structured CSV data using natural language, with a focus on cybersecurity vulnerabilities. The system integrates Python, LangChain, and SQLAlchemy to process, analyze, and provide detailed insights into security risks.

The primary features include:

  • Real-time querying of CSV data through natural language.
  • Integration with OpenAI GPT for intelligent and context-aware responses.
  • Support for SQL databases, making the solution scalable for larger datasets.
  • Automatic report generation for detailed vulnerability descriptions.

Key Features

  • Natural Language Querying: Users can ask questions in plain English, and the system generates precise, data-driven responses.
  • Cybersecurity Focus: Designed specifically for analyzing vulnerabilities and generating comprehensive reports.
  • Report Generation: Automatically generates detailed, formatted reports in text files for easy sharing.
  • Database Integration: Stores and queries CSV data using SQLite for scalability and reliability.
