Project Overview

"Going for Gold" is an innovative web-based trivia quiz app designed to test and entertain users through dynamic questions and a sleek interface. The project highlights include:

  • Developed using Node.js, Express.js, and MongoDB.
  • Hosted on AWS EC2 with additional deployment on Raspberry Pi.
  • Deployed in Salle Serveur for testing under real-world conditions.
  • Integrated PM2 for process management and NGINX for reverse proxy setup.
  • Real-time multiplayer capabilities with dynamic score tracking.
